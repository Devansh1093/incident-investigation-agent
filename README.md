# Incident Investigation Agent

Hackathon project that automates software incident investigation.

## Features

- GitHub integration
- Log analysis
- Metrics analysis
- Slack integration
- Timeline generation
- Root Cause Analysis (RCA)
- Recommendation engine

## Tech Stack

- Python
- FastAPI
- Coral
- REST APIs
- JSON
- GitHub
